云注册开放API说明文档

1.简介

        客户在产品注册系统完成产品注册， 系统将注册信息推送到云平台。

2.流程说明

3.URL地址

        /openapi/service/product/register

4.请求方式

        HTTP   POST

5.参数说明

        5.1请求参数 :Json对象包含以下字段

 参数 数据类型 是否必填 说明
 id string 是注册记录id（友互通提供）
 companyId string 是企业ID（有互通提供）
 productlineId string 是产品线ID（有互通提供） 
 productCode     string 是产品条码
 productName string 是产品名称
 activeCode string 是激光码
 serviceCode string 是服务识别码
 oldActiveCode string 否激光码（旧），多个以“|”分隔
 oldProductCode string 否对应的旧的产品条码

      5.2响应数据

        成功则返回：

{"result": "success"}

        失败详见文档最后响应消息统一说明
