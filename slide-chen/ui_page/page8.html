AngularJS运用之——常用指令

1.样式控制指令:ng-class用来给元素绑定类名，其表达式的返回值可以是以下三种：

（1）类名字符串:可以用空格分割多个类名，如"class1 class2"；

（2）类名数组:数组中的每一项都会层叠起来生效；

（3）一个名值对应的map。其键值为类名，值为boolean类型，当值为true时，该类会被加在元素上。如下：

<li ng-click="isClick='/page1'"
       ng-class="{selected:'/page1'==isClick}">
            <a ui-sref="page1">
            <img src="images/thumbs/main.png" alt="1"/>
            </a>
    </li>

如果你想拼接一个类名出来，可以使用插值表达式，如：<div class="{{style}}text">字体样式测试</div>

与ng-class相近的，ng还提供了ng-class-odd、ng-class-even两个指令，用来配合ng-repeat分别在奇数列和偶数列使用对应的类。这个用来在表格中实现隔行换色再方便不过了。

2.ng-show，ng-hide

ng-show和ng-hide的值为boolean类型的表达式，当值为true时，对应的show或hide生效。框架会用display:block和display:none来控制元素的显隐。

3.表单控件功能相关的:取值均为boolean类型，当值为true时相关状态生效。

（1）ng-checked控制radio和checkbox的选中状态

（2）ng-selected控制下拉框的选中状态

（3）ng-disabled控制失效状态

（4）ng-readonly控制只读状态

4.事件绑定相关:在模板中可以用变量$event将事件对象传递到controller中。

（1）ng-change="change($event)"

用于告诉 AngularJS 在 HTML 元素值改变时需要执行的操作。

（2）ng-click="handleClick($event)"

在 HTML 元素被点击时需要执行的操作。

5.ng-src和ng-href

由ng的启动及执行过程可知，ng框架是在DOMcontent加载完毕后才开始发挥作用。假如我们模板中有一张图片如下：

<img src="{{imgUrl}}"/>

那么在页面开始加载到ng编译完成之前，页面上会一直显示一张错误的图片，因为路径{{imgUrl}}还未被替换。

为了避免这种情况，我们使用ng-src指令，这样在路径被正确得到之前就不会显示找不到图片。同理，<a>标签的href属性也需要换成ng-href，这样页面上就不会先出现一个地址错误的链接。

6.ng-if

ng-if指令可以根据表达式的值在DOM中生成或移除一个元素。如果赋值给ng-if的表达式的值是false，那对应的元素将会从DOM中移除，否则生成一个新的元素插入DOM中。ng-if同no-show和ng-hide指令最本质的区别是，它不是通过CSS显示或隐藏DOM节点，而是删除或者新增结点。

ng-if重新创建元素时用的是它们编译后的状态。此时这些元素的状态会是它们的原始状态。而ng-show和ng-hide则可以保留dom元素上次修改后的状态。

7.ng-model:

绑定了HTML表单元素到scope变量中。如果 scope 中不存在变量, 将会创建它。

ng-modle是$scope <-> view的双向绑定。举个例子：

<div class="box">
        <p>images目录中已有：img1-img3可测试!</p>
        <label for="myInput">输入图片名称：</label>
        <input id="myInput" type="text" ng-model="imgName"/>
    </div>
    <div class="img-box">
        <img ng-src="images/{{imgName}}.jpg">
            Hello  {{imgName}}!
        </img>
    </div>

我们通过ng-model将图片的地址和输入框的值绑定，这样，当我们输入框的值变化的时候，通过打印输出imgName的值和图片地址的变化，我们可以知道，ng-model实现了view和scope之间的双向绑定。例子请点这里

8.ng-bind:

ng-bind指令告诉AngularJS使用给定的变量或表达式的值来替换HTML元素的内容。

ng-bind是从$scope -> view的单向绑定。

9.ng-repeat:

ng-repeat指令用于循环输出指定次数的 HTML 元素。特别适用于列表和表格等可循环生成元素。举个例子：

<table ng-controller="myCtrl" border="1">
        <thead>
            <tr>
                <th>name</th>
                <th>Country</th>
            </tr>
        </thead>
        <tbody>
            <tr ng-repeat="x in records">
                <td>{{x.Name}}</td>
                <td>{{x.Country}}</td>
            </tr>
        </tbody>
    </table>

ng-repeat会将模版按照模型数据量循环生成到页面上。比如例子中records模型为：

app.controller("myCtrl", function($scope) {
        $scope.records = [
            {"Name" : "Aimy", "Country" : "Germany"},
            {"Name" : "John", "Country" : "Sweden"},
            {"Name" : "Moctezuma", "Country" : "Mexico"},
            {"Name" : "Handel", "Country" : "Austria"}
        ]
        });

生成结果如下：

name Country
{{x.Name}} {{x.Country}}
